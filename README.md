# SQL Sales Data Analysis Project

## Project Overview
This project simulates a sales database and demonstrates core SQL skills used in data analytics. It includes table creation, data insertion, JOIN operations, GROUP BY aggregation, and analytical queries to extract business insights.

## Objectives
- Create relational database tables
- Insert structured sales data
- Analyze customer purchase behavior
- Calculate total and average sales metrics

## Tools Used
- SQL
- Relational Database Concepts

## Key Skills Demonstrated
- Writing JOIN queries
- Using GROUP BY and aggregate functions (SUM, AVG)
- Data modeling and table relationships
- Query-based business analysis

## Example Insights Generated
- Total sales by city
- Average order value
- Customer purchase breakdown

## Why This Project Matters
Understanding structured query language (SQL) is foundational for data analytics roles. This project demonstrates my ability to design tables, manipulate data, and extract meaningful insights from relational datasets.
